# Historias de Usuario (AquaKeeper)

- **US-001**: Como dueño, quiero registrar clientes y sus piletas con litros y estado (pH, turbidez, algas) para decidir acciones.
- **US-002**: Como técnico, quiero calcular dosis de productos según estrategia (mantenimiento/choque/antisarro/alguicida) para no sobredosificar.
- **US-003**: Como dueño, quiero ver el % de cobertura de productos que el cliente tiene en su casa vs lo requerido hoy para saber qué falta.
- **US-004**: Como técnico, quiero registrar visitas y saber si “debo ir” hoy en base a estado del agua y faltantes.
- **US-005**: Como dueño, quiero alertas de stock mínimo (local y cliente) para reponer a tiempo.
- **US-006**: Como dueño, quiero persistir un resumen de piletas y visitas para control histórico.
